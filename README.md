# Happy Mandado Webapp

## Development setup

### Requirements
 - Node
 - React Scripts

### Load configuration
```
$ source .env
```

### Update local hosts
```
127.0.0.1    happymandado.local
127.0.0.1    service.happymandado.local
127.0.0.1    auth.happymandado.local
```

### Install
```
$ npm i
```

### Run
```
$ npm run start
```
